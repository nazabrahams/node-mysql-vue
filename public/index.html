Path Method Description
Users Table
/users/ GET Gets all of the users
/users/:id GET Gets a single by id
/users/ POST Adds a user
/users/:id DELETE Deletes a by id
/users/:id PATCH Edits all information of a user by id
Fruit Table
/fruit/ GET Gets all of the fruit
/fruit/:id GET Gets a single fruit by id
/fruit/ POST Adds a fruit
/fruit/:id DELETE Deletes a fruit by id
/fruit/:id PATCH Edits all information of a fruit by id
